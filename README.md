# 伊吹しろう Official Website

VTuber「伊吹しろう」の公式サイトです。

## プロジェクト概要

本サイトは、VTuber「伊吹しろう」の公式情報を発信するポータルサイトです。
プロフィール、実績、グッズ、ガイドラインなど、伊吹しろうに関する情報をまとめています。

## プロジェクト構造

```
/
├── index.html              # トップページ
├── profile.html            # プロフィールページ
├── achievements.html       # 実績ページ
├── goods.html             # グッズページ
├── guidelines.html        # ファンガイドライン
├── faq.html               # よくある質問
├── contact.html           # お問い合わせ
├── terms.html             # 利用規約
├── privacy.html           # プライバシーポリシー
│
├── /css                   # スタイルシート
├── /js                    # JavaScript
├── /data                  # JSONデータ
└── /assets                # 画像・リソース
```
